Cuestionario Ejercicio numero 1 HTML

  • Dar un par de ejemplos de uso de etiquetas meta
  • Nombrar al menos 2 atributos aplicables a la etiqueta HTML
  • ¿Como se relaciona un header http con la etiqueta head de HTML?
  • ¿Que es UTF8?
  • ¿Que atributo HTML sirve para aplicar estilo ¿A que etiqueta es aplicable?
  • ¿Donde se indica que un enlace debe ubicar el documento en una nueva pestana?
  • Explicar cuando usarian etiquetas semanticas y con que proposito.
  • ¿Que atributos de estilo basicos aplicarian a una tabla? Explicar
  • ¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?
  • ¿Que atributos marcan la agrupacion de celdas de una tabla?.¿A que etiquetas se aplican?




A continuacion se detallan las respuestas por orden:


  • Meta Keywords: Buscador de palabras Clave
    Meta Autor: Refieren al autor o disenador de una pagina web.

  • Un atributo de la etiqueta HTML es "lang" que indica el lenguaje, "class" establece la clase CSS que se aplica a los estilos del elemento, "id" es un identificador del elemento, "style" establece el estilo CSS

  • La que la etiqueta "header" de html refiere a un encabezado del documento que contiene datos y el header de http puede utilizarlo varias veces

  • El Unicode UTF-8 es un formato de caracteres en el que cada caracter esta representado por 8 bits. Permite representar varios idiomas incluyendo caracteres de utilizados en lenguajes asiaticos.

  • Para aplicar estilo en html se utiliza el atributo "style" , que permite especificar un estilo CSS para cambiar su formato de presentacion. El atributo es Global, por lo tanto se puede aplicar a cualquier etiqueta html

  • Se indica dentro de la etiqueta "a" con el atributo "target" ademas de la referencia al documento

  • Se utilizan cuando se debe dar un significado a las partes de un documento html, como puede ser una frase o palabra. Un ejemplo es la etiqueta "strong"

  • Para visualizar mejor una tabla, se pueden utilizar los siguientes atributos:
    - align: Para alinear el texto de la tabla
    - background: Para cambiar el color de fondo de la tabla, por ejemplo para resaltar algunas celdas.
    - bordercolor: Para resaltar el principio y fin de la tabla
    - border: Para realizar las lineas mas gruesas o finas.

  • La etiqueta "body" se utiliza para determinar principio y fin del contenido de un documento HTML, en cambio en el protocolo HTTP el "body" lleva los datos por ejemplo, ingresados en un formulario.

  • Existen dos atributos que permiten agrupar celdas:
    -Colspan: Agrupa y desplaza celdas a la derecha
    -Rowspan: Agrupa y desplaza celdas hacia abajo
    Se pueden utilizar en conjunto, son atributos de la etiqueta "td"
